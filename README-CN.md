# Github stars 管理器

这是一个用来管理你的github stars的网页工具，它通过标签来管理和分类你的stars。所有的数据保存在你自己的github gists.

# Get your personal access token

从 [这里](https://github.com/settings/tokens) 获取 access token

如果你需要长期使用，过期时间请选择 No expiration
